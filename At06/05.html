MUDAR ELEMENTOS DE UMA ARRAY

Para mudar o valor de um elemento específico, refira o número index: 

            String[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
            cars[0] = "Opel";
            System.out.println(cars[0]);
--->Agora o output será Opel ao invés de Volvo
